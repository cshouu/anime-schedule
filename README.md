# anime-schedule
这是一个Chrome插件，用于从相关网站上获取动漫时间表、动漫更新、微博热搜、美剧时间表、热映电影信息

下载源码后，在Chrome内核的浏览器（Chrome、QQ浏览器、搜狗浏览器等）的开发者模式下添加文件夹
